# Actividad-1-Inteligencia-Artificial-UEM

## Datos NGMEP - MUNICIPIOS.csv (uso en práctica universitaria)

En este repositorio se aloja el fichero "MUNICIPIOS.csv" contenido originalmente en el archivo BD_Municipios-Entidades.zip, correspondiente al Nomenclátor Geográfico de Municipios y Entidades de Población (NGMEP), edición 2024.

Estos datos son publicados por el Instituto Geográfico Nacional (IGN) y distribuidos a través del Centro Nacional de Información Geográfica (CNIG). 

El uso es para una práctica de universidad, en concreto la descarga automática del archivo desde un cuaderno de Google Colab

**Fuente original:** La fuente original del fichero es del Centro de Descargas del CNIG/IGN (NGMEP 2024)
**Licencia de datos:** El CNIG indica que la licencia de uso es compatible con **CC-BY 4.0** (requiere atribución al IGN/CNIG)
